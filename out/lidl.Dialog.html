Class: Dialog

lidl.Dialog(token, targetElementSelector, action, type, content, args, callback)

new Dialog(token, targetElementSelector, action, type, content, args, callback)

Constructor of dialog object
Parameters:
Name Type Description
token lidl token
targetElementSelector selector for the element[s] to trigger the dialog
action function to execute when dialog is confirmed
type type of dialog box. Currently supported: confirmDelete, imageUpload, removeDBKey, addDBKey
content JSON object containing the following data: title, message, confirmMessage
args function to be executed when calling the action function. Use this to access information from inside the dialog function.
callback Object callback functions for dialog interaction. Currently supported: onCancel, onConfirm
